# djangogram 프로젝트

## 주제

djangogram - Django 웹 애플리케이션을 통한 소셜 미디어 스타일의 사진 공유 플랫폼 구현

## 작성자

이영직

## 사용 툴

Visual Studio Code

## 사용 언어

Django

## 목적

이 프로젝트는 Django 웹 프레임워크를 활용하여 소셜 미디어 플랫폼을 구현함으로써 웹 개발의 기본 개념을 이해하고, 프론트엔드와 백엔드 간의 상호작용을 경험합니다.
